# canyon-cli

[![Continuous integration](https://github.com/canyon-network/canyon-cli/actions/workflows/ci.yml/badge.svg)](https://github.com/canyon-network/canyon-cli/actions/workflows/ci.yml)

A CLI tool for interacting with [canyon network](https://github.com/canyon-network/canyon) based on [substrate-subxt](https://github.com/paritytech/substrate-subxt).

## Usage

```bash
$ cargo build --release
$ ./target/release/canyon-cli --help
```
